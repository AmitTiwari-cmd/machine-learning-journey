# Linear Regression

## Purpose

Used to predict continuous numerical values.

Examples:

* House Price Prediction
* Salary Prediction
* Sales Forecasting

## Type

Regression Algorithm

## Input & Output

X = Independent Features (Area, Bedrooms, Age)

y = Numerical Output (Price)

## Key Idea

Finds the best-fit line that minimizes prediction error.

## Workflow

1. Import Dataset
2. Data Preprocessing
3. Train-Test Split
4. Train Model using fit()
5. Predict using predict()
6. Evaluate Model

## Evaluation Metrics

* R² Score
* MAE
* MSE
* RMSE

## Advantages

* Simple
* Fast
* Easy to interpret

## Limitations

* Assumes linear relationship
* Sensitive to outliers

## Project Completed

* House Price Prediction

## Interview Questions

Q. Regression or Classification?
→ Regression

Q. Why?
→ Output is continuous (price).
