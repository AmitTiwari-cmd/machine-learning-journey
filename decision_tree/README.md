# Iris Flower Classification using Decision Tree

## Objective
Classify iris flowers into species using a Decision Tree Classifier.

## Dataset
Iris Dataset

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Target
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Concepts Learned
- Decision Trees
- Entropy
- Overfitting
- Hyperparameter Tuning
- Confusion Matrix

## Hyperparameter Tuned
max_depth

## Observation
Depth 3 achieved the best performance on the dataset.

## Libraries Used
- Pandas
- Scikit-Learn
- Matplotlib
