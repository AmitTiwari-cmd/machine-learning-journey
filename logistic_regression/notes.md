# Logistic Regression

## Purpose

Used for Classification Problems.

Examples:

* Survive / Not Survive
* Pass / Fail
* Spam / Not Spam

## Type

Classification Algorithm

## Output

Probability between 0 and 1.

Example:
0.85 → Survived

0.10 → Not Survived

## Key Idea

Uses Sigmoid Function to convert predictions into probabilities.

## Workflow

1. Data Cleaning
2. Handle Missing Values
3. One-Hot Encoding
4. Train-Test Split
5. Train Model
6. Predict Classes
7. Evaluate

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Important Concepts

### One-Hot Encoding

Used when feature is categorical.

Example:

Location:

* Delhi
* Mumbai
* Lucknow

Converted into numerical columns.

### Overfitting

High Training Accuracy
Low Testing Accuracy

Example:
Train = 99%
Test = 58%

Model memorized data.
